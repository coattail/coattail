## What I Build

- Macro dashboards for long-range country comparison and signal tracking
- Housing and multi-asset tools that make price regimes easier to read
- Research-first visual products for valuation, holdings, and market narrative work

## Featured Work

### [Macro-Dashboard](https://github.com/coattail/Macro-Dashboard)
Static dashboard for comparing countries across growth, trade, fiscal, and demographic data with export-ready visuals.

### [Multi-Asset-Dashboard](https://github.com/coattail/Multi-Asset-Dashboard)
Research workspace for housing, commodities, equities, and cross-asset comparisons on one visual timeline.

### [Centaline-Leading-Index](https://github.com/coattail/Centaline-Leading-Index)
China secondary housing tracker with dual-source comparison, drawdown analysis, and clean export flows.

### [guru-13f-monitor](https://github.com/coattail/guru-13f-monitor)
SEC 13F tracker for following institutional portfolios, quarter changes, and crowding signals.

## Current Focus

- Building calmer interfaces for dense market data
- Turning recurring data refreshes into reliable GitHub workflows
- Packaging research outputs so they are presentation-ready by default

## Design Notes

This profile uses lightweight SVG motion instead of heavy widgets:

- slow grid drift and signal sweeps in the hero
- chart-line draw animations in project cards
- one live activity module instead of many competing stat blocks
